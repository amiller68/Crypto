Basic Language Constructs

Things that you should learn about any language when you first encounter it.

  • Loops
  • Conditionals
  • Functions
  • Useful data structures and the runtime complexities of common operations on them
  • Strings
  • Lists
  • Dictionaries
  • Tuples
  • Reading and writing files
  • Converting between chars and ints (ord and chr)
  • How to run a Python script (if __name__ == '__main__':)
  • Play with a Python shell, i.e., run python on the commandline  (I highly recommend using ipython instead of the native Python shell because it  has lots of added features for your convenience. Not a requirement by any measure.)
  • Know how to install and import Python packages (using pip)

Note: You will probably not need to know much about object-oriented programming in Python (phew!), other than really simple things, such as student.grade means accessing the grade attribute of an object student.  Just like a struct in C if you're more used to that.

Manipulating Bytes

  • For our purposes, you will almost always want to use bytes objects (i.e., b'hi' instead of 'hi')
  • Strings and bytes in Python are not mutable. To have a mutable object,  you will want to use a bytearray (https://docs.python.org/3/library/functions.html#func-bytearray). This way, you can do things like a[0] = a[0] + 1, just like you would in C!

Base 64

Sometimes, you will need to encode strings in base 64, and you can use  the handy base64 library: https://docs.python.org/2/library/base64.html 

Plotting

  • You will want to know how to plot simple things, such as histograms for frequency analysis
  • matplotlib.pyplot is the most popular choice, especially because of its low barrier  to entry (it's simple, but not fancy). That said, you're welcome to use any plotting library of your choice.
  • To install matplotlib on macOS, go to the terminal and type:
  • $ cd /usr/local/bin
  • $ python3.7 -m pip install matplotlib
  • If you have another OS, you can find similar install instructions here: https://ehmatthes.github.io/pcc/chapter_15/README.html 

Other Convenient Things to Know

These are things that you don't need to know, but will make your life easier. 

Python provides many built-in features which make prototyping small scripts really quick and easy.  Here are some of these features you might want to look at eventually:

  • set and set operations
  • range, enumerate, zip, and other ways to iterate
  • collections.Counter, collections.defaultdict, collections.OrderedDict
  • itertools.product, itertools.permutations, itertools.combinations, itertools.combinations_with_replacement
